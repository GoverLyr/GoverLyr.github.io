HTML

Viewport

width：控制 viewport 的宽度，可以指定的一个值，如 600，或者特殊的值，如 device-width 为设备的宽度。
initial-scale：初始缩放比例，也即是当页面第一次加载的时候缩放比例。
maximum-scale：允许用户缩放到的最大比例。
minimum-scale：允许用户缩放到的最小比例。
user-scalable：用户是否可以手动缩放。

1
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">

input行内事件

onkeyup：事件会在键盘按键被松开时发生。
onkeydown：事件会在用户按下一个键盘按键时发生。
onfocus：事件在对象获得焦点时发生。
onblur：事件会在对象失去焦点时发生。

1
2
3
4
<input type="text" onkeyup="console.log('松开了键盘')">
<input type="text" onkeydown="console.log('按下了键盘')">
<input type="text" onfocus="console.log('光标进入了输入框，内容：'+this.value)" value="value">
<input type="text" onblur="console.log('光标离开了输入框，内容：'+this.value)">
