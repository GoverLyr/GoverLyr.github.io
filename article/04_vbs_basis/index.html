VBS脚本基础教程：编写及运行VBS代码
2019-05-11 0

VBS是VBScript的简称，是由微软推出的一套基于Windows的脚本语言，该脚本无需编译成EXE的可执行文件。
个人感觉和bat挺像的。能够做挺多事情的。

创建VBS脚本文件

其实挺简单的，是个文本编辑器都能创建。
最简单方法就是新建一个TXT文本，之后把文件后缀改成 .vbs 即可。
VBS的文件编码是GB2312的，使用UTF-8的话，中文会乱码。

VBS之Hello World！

将以下代码复制到TXT文本，修改文本后缀即可运行啦。 (￣▽￣)／

1
2
3
Dim str
str = "Hello World"
Msgbox(str)

变量的定义与使用

VBS存储数据的量分为常量和变量两种。
顾名思义，常量指在整个程序执行过程中保持不变的量，通常用于存储一些固定不变的数值。而变量则可以在程序执行过程中不断被改变的量，用于实现数据的读取和存储操作。

常量

对于常量的定义，我们可以使用Const关键字来定义，例如：π

1
const PI=3.1415926

按约定俗成的定义，常量全部使用大写字母来表示，以示区别。

变量

通常可以使用DIM来定义变量，建议在定义变量的同时指定数据类型。由于VBS为非严格型变量定义，因此可以不指定数据类型，而由系统自己的识别。

1
2
3
4
Dim i,j,k
i=3
j=4 
k=5

VBS输入框

弹出输入框，可与用户进行交互。

1
2
3
Dim name 
name = Inputbox("请输入您的名字：")
Msgbox name
