Hexo常用命令

Hexo 有几十个命令，能够经常使用的却不多。
这里就简单列几个常用的命令。

hexo server

可以启动本地服务器，用于预览主题。默认地址：http://localhost:4000
hexo server 可缩写为 hexo s

1
2
$ hexo s
$ hexo server

hexo new

可以执行下列命令来创建一篇新文章。
hexo new 可缩写为 hexo n

1
2
$ hexo n "标题"
$ hexo new "标题"

hexo deploy

可以自动生成网站静态文件，并部署到设定的仓库。
hexo deploy 可缩写为 hexo d

1
2
$ hexo d
$ hexo deploy

hexo generate

可以生成网站静态文件到默认设置的 public 文件夹里。
hexo generate 可缩写为 hexo g

1
2
$ hexo g
$ hexo generate

hexo clean

既然可以生成文件，那当然也可以删除。可以删除缓存文件 db.json 和已生成的静态文件 public 。

1
$ hexo clean

更多详细可以进入Hexo官网查看，https://hexo.io
